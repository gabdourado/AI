# Convenções de Notação Matemática
> Sumário de símbolos, terminologias e convenções tipográficas adotadas nos notebooks de Machine Learning.
---
## Convenções Tipográficas Gerais
| Tipo | Notação | Exemplo |
|---|---|---|
| Tamanho de um conjunto de Dados | Letra maiúscula | $N, M, K$ |
| Dimensão de um atributo do conjunto de Dados | Letra maiúscula | $D$ |
| Conjunto | Letra cursiva maiúscula | $\mathcal{D}, \mathcal{C}, \mathcal{F}$ |
| Vetor coluna | Letra minúscula em negrito | $\mathbf{x}, \mathbf{w}, \boldsymbol{\mu}$ |
| Matriz | Letra maiúscula em negrito | $\mathbf{X}, \mathbf{W}, \mathbf{S}$ |
| Escalar real | Letra minúscula em itálico | $y_i, \eta, \lambda$ |
| Índice de amostra | Subscrito inteiro | $\mathbf{x}_i,\ i = 1, \dots, N$ |
| Índice de dimensão | Subscrito inteiro | $x_j,\ j = 1, \dots, D$ |
| Estimador / valor predito | Acento circunflexo (*hat*) | $\hat{\mathbf{w}},\ \hat{y},\ \hat{c}$ |
| Parâmetro em iteração $t$ | Sobrescrito entre parênteses | $\mathbf{w}^{(t)},\ \boldsymbol{\mu}_i^{(t)}$ |
| Função de custo / perda | Letra cursiva maiúscula | $\mathcal{L}(\mathbf{\cdot})$ |
| Função indicadora | Símbolo blackboard bold | $\mathbb{1}(\cdot)$ |
| Espaço vetorial real de dimensão $D$ | Símbolo blackboard bold | $\mathbb{R}^D$ |
| Norma euclidiana | Duplas barras verticais | $\lVert \mathbf{x} - \boldsymbol{\mu} \rVert$ |
| Gradiente em relação a $\mathbf{w}$ | Nabla com subscrito | $\nabla_{\mathbf{w}}\,\mathcal{L}(\mathbf{w})$ |
| Transposta de matriz/vetor | Sobrescrito $\top$ | $\mathbf{X}^\top,\ \mathbf{x}_i^\top$ |
| Pseudoinversa de Moore-Penrose | Sobrescrito $\dagger$ | $\mathbf{X}^\dagger$ |
| Vizinhança de $k$ vizinhos de $\mathbf{z}$ | Notação de conjunto com subscrito | $N_k(\mathbf{z})$ |
| Média condicional à classe $c$ | Subscrito duplo $(c, i)$ | $\mu_{c,i}$ |
| Variância condicional à classe $c$ | Subscrito duplo com expoente | $\sigma_{c,i}^2$ |
